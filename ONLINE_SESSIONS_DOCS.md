## 🎬 فلوی استفاده

### مشاور:
```
1. لاگین → منوی "جلسات آنلاین"
2. کلیک "ایجاد جلسه آنلاین جدید"
3. پر کردن فرم:
   - عنوان (مثلاً: کلاس ریاضی جمع‌بندی)
   - تاریخ، ساعت، مدت
   - انتخاب دانش‌آموزان (چک‌باکس)
   - تنظیم دسترسی‌ها (میکروفون، دوربین، ...)
4. کلیک "ایجاد جلسه"
5. در زمان جلسه → کلیک "شروع جلسه"
6. وارد اتاق می‌شوید
7. میزبان کنترل کامل دارد:
   - چت، تخته، واکنش، دست
   - مشاهده لیست شرکت‌کنندگان
   - مشاهده دست‌های بلند شده
   - پایان جلسه
```

### دانش‌آموز:
```
1. اعلان: "جلسه آنلاین فعال" روی داشبورد
2. کلیک "ورود به اتاق"
3. (اگر میزبان اجازه داده) میکروفون و دوربین فعال می‌شود
4. می‌تواند:
   - چت کند
   - از تخته استفاده کند (اگر فعال باشد)
   - دست بلند کند
   - واکنش بفرستد
```
